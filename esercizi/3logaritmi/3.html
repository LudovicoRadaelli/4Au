Proprietà dei logaritmi

\[ \log_{b}{\left(a\right)} \color{red}{\,+\,} \color{black}{} \log_{b}{\left(c\right)} = \log_{b}\left(a \color{red}{\,\cdot\,}\color{black}{}c\right) \]
\[ \log_{b}{\left(a\right)} \color{red}{\,-\,} \color{black}{} \log_{b}{\left(c\right)} = \log_{b}\left(\color{red}{\dfrac{\color{black}{a}}{\color{black}{c}}}\color{black}{}\right) \]
\[ \log_{b}\left(a^{\color{red}{c}}\color{black}{}\right) = \color{red}{c} \color{black}{} \log_{b}\left(a\right) \]
\[ \log_{b}{\left(a\right)} = \dfrac{\log_{\color{red}{c}}{\left(a\right)}}{\log_{\color{red}{c}}{\left(a\right)}} \]

Equazioni logaritmiche per passaggio agli argomenti

Prima di tutto impariamo a riconoscere equazioni logaritmiche di questa tipologia.

Esse presentano in entrambi i membri somme e sottrazioni di logaritmi o di loro multipli.
Di conseguenza per questa tipologia possiamo usare le proprietà dei logaritmi studiate nel paragrafo 2.

Facciamo alcuni esempi:


  • \( log\left(x\right) - 3\,log\left(x\right) = log\left(x\right) \)


  • \( 3\,log\left(x\right) + 5\,log\left(2x - 5\right) = log\left(x\right) \)


  • \( log\left(x\right) + 2\,log\left(x + 1\right) = log\left(x - 1\right) - log\left(2x\right) \)


  • \( log_{_{3}}\left(x\right) + \color{red}{2}\color{black}{} = \color{blue}{0} \)

    perché possiamo trasformare l'equazione in

    \( log_{_{3}}\left(x\right) + \color{red}{log_{_{3}}\left(9\right)}\color{black}{} = \color{blue}{log_{_{3}}\left(1\right)} \)


Nel prossimo esempio vediamo la strategia da mettere in campo per risolvere un'equazione di questo tipo.


Esempio

In generale possiamo dire che;

Per risolvere un'equazione del tipo "passaggio agli argomenti":


  • si effettuano le condizioni di esistenza;

  • si usano le proprietà dei logaritmi fino a riscrivere l'equazione nella forma \[ log\left(\color{red}{}A\left(x\right)\color{black}{}\right) = log\left(\color{blue}{}B\left(x\right)\color{black}{}\right) \] Successivamente si deduce che \[ \color{red}{}A\left(x\right)\color{black}{} = \color{blue}{}B\left(x\right) \] (passaggio agli argomenti )

  • Si risolve l'equazione \[ A\left(x\right) = B\left(x\right) \] e si controlla se le soluzioni ottenute appartengono al campo di esisteza




Esercizio


\[ log_{_2}(x+1) + log_{_2}(x-2) = 2 + log_{_2}(x-1) \]
Soluzione: L'equazione è risolta per \(\,\,\,x = 1 + \sqrt{7}\).
(La soluzione \(1 - \sqrt{7}\) non è accettabile per le C.E.)




Esercizio


\[ ln(2x - 1) + ln(x + 3) = ln(x + 1) + ln(2x + 1) \]
Soluzione: L'equazione è risolta per \(\,\,\,x = 2\).




Esercizio


\[ \frac{1}{2}log(x^2 + 7x + 12) - log(2) = log(x + 3) \]
Soluzione: \(\,\,\,x = -\frac{8}{3}\)
